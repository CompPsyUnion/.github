# CPU介绍

宁波诺丁汉大学计算机爱好者协会，英文名：Computer Psycho Union

该Github组织成立于2024年9月，非社团成立时间。

## 如何加入
通常情况下，自每年9月下旬的百团大会起开始招新，10月公布面试结果并迎新。

后续想加入也可直接联系核心成员进行交流。

## 关于项目仓库发起或转移的要求
需要有明确的规划，比如：主要使用的语言，项目readme等。不能烂尾。
